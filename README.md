##  Getting Started

### Prerequisites

- Node.js (v14 or newer)
- npm or Yarn

### Installation & Running Locally

```bash
# Clone the repo
git clone https://github.com/your-username/sankalp-clone.git
cd sankalp-clone

# Install dependencies
npm install
# or
yarn install

# Start dev server
npm run dev
# or
yarn dev

# Visit: http://localhost:3000 (or Vite default port)
